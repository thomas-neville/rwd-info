Responsive Web Design Information

Responsive Web Design adjusts the content of a webpage based on the display size of the device viewing it. These adjustments make the content easier to view and can also improve navigation and functionality. For example, a webpage may have several columns of text and pictures that can be easily read and seen on a large desktop monitor. When the same page is viewed on a smartphone the text and images will shrink down to fit on the smaller screen of the phone making it difficult to see. Responsive Web Design solves this by reorganizing the content with things like breakpoints.
